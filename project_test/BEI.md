# Comandos do BEI (BEInit)

- bei init (Apenas para iniciar projeto)
- bei run (inicia o servidor com php, mariadb e bun)
- bei stop (encerra o servidor)
- bei php (executa comandos do php)
- bei bun (executa comandos do bun)
- bei composer (executa comandos do composer)
- bei status (mostra o status de tudo em execussão na ferramenta BEI)

## Por que usar?

ao invés de criar diversas imagens no docker e consumir toda a RAM. Esse gerenciador é extremamente simples focado exclusivamente em php com mariadb,
você pode simplesmente usá-lo para iniciar e gerenciar seu servidor php com mariadb.
